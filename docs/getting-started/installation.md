# Installation

You can run mf-doctor as a project dependency (recommended for CI) or one-off via npx.

| Method                           | Command                |
| -------------------------------- | ---------------------- |
| Project dependency (recommended) | `npm install -D mf-doctor` |
| One-off run                      | `npx mf-doctor analyze .`  |

After installing as a dependency, run:

```bash
npx mf-doctor analyze .
```

See [Quick Start](/docs/getting-started/quick-start) for your first analysis.
